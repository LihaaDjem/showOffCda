Plan de Travail Agile/Scrum

Durée totale du projet : 12 semaines (3 mois)

Sprint 0 (Semaine 1) : Préparation et conception

  • Objectif : Définir les user stories, créer les wireframes et maquettes.
  • Tâches :
    • Réunion de lancement avec le client pour valider les besoins.
    • Création des user stories pour chaque type d'utilisateur (coursier, client, commerçant, administrateur).
    • Conception des wireframes et maquettes sur Figma.
    • Validation des maquettes par le client.

Sprint 1 (Semaines 2-3) : Développement de l'interface utilisateur

  • Objectif : Développer les interfaces utilisateur pour les clients et les coursiers.
  • Tâches :
    • Intégration des maquettes Figma en HTML/CSS avec Angular.
    • Développement des pages d'accueil, de connexion et d'inscription.
    • Intégration de Google Maps pour la géolocalisation.
    • Mise en place de Firebase Authentication pour la gestion des utilisateurs.
